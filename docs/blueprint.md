# **App Name**: PaisaPathshala

## Core Features:

- Virtual Portfolio Simulator: Simulate investments in Indian stocks and mutual funds with virtual money.
- Gamified Learning Modules: Interactive lessons on budgeting, SIPs, UPI, taxes, and credit scores.
- India-Centric Content: Content contextualized for the Indian financial ecosystem, relevant to the target audience.
- Interactive Budgeting Tools: Track income, categorize expenses, and set financial goals. tool for assisting with financial projections
- Fraud Identification Challenges: Scenarios to identify and avoid digital financial frauds.
- Personalized Financial Advice Tool: An AI powered tool that offers personalized financial tips based on user's simulation data and learning progress

## Style Guidelines:

- Primary color: Saturated blue (#2962FF) to convey trust and security.
- Background color: Light blue (#E5E8EF), a very desaturated version of the primary, for a clean, calming backdrop.
- Accent color: Purple (#9D2EC5) to highlight important interactive elements and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for headlines or body text
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean, modern icons to represent different financial concepts and actions.
- Subtle animations to enhance user engagement and guide users through the app.
